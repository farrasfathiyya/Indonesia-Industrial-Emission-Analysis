# Cilegon Industrial Emission Mapping
This project is a preliminary data analysis tool developed to support research on 
industrial decarbonization pathways in Indonesia, specifically focusing on the 
Cilegon industrial cluster.

## Features
- Geospatial mapping of major industrial emission sources.
- Data processing using Python (Pandas) and interactive visualization (Folium).
- Future scenario modeling for Net Zero 2060 targets.
